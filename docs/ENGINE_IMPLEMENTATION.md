# Engine implementation baseline

- Язык: **C++17**.
- Кроссплатформенность: поддерживает **Windows, Linux, macOS** (при наличии совместимых зависимостей).
- Структура проекта: используется базовый каркас с папками `src/`, `include/`, `assets/`.

Этот документ фиксирует базовые решения для модулей `ResourceManager`, `Scene`, `Renderer`, `PhysicsSystem`, `InputManager`, `AudioManager`, `SceneManager` и тестов GoogleTest.
