# План развития COre-GOLIAF (обновлено)

## Базовые решения
- Язык ядра: **C++17**.
- Платформы: **Windows, Linux, macOS**.
- Билды: **Dev**, **Player**, **LanServer**.
- Текущий рендер по умолчанию: модульный каркас (`Renderer`), с дальнейшим расширением под OpenGL/Vulkan/DirectX/Metal через backend-слой.

## Что исправлено после ревью
1. `ResourceManager`:
   - Добавлены API-алиасы `LoadTexture/LoadModel/LoadSound`, `GetResource`, `ReleaseResource`.
   - Кэш теперь использует нормализованные пути (`lexically_normal`) для исключения дублей `a/../a/file`.
   - Добавлена потокобезопасность (`std::mutex`).
2. `Renderer`:
   - Улучшена загрузка «программ шейдера»: проверяется существование файлов и непустое содержимое.
   - Ведётся реестр загруженных программ (`hasShaderProgram`).
   - `drawMesh` валидирует наличие шейдера материала.
3. Сборка:
   - Исправлено dev-warning CMake `CMP0135` для `FetchContent` (включён `DOWNLOAD_EXTRACT_TIMESTAMP TRUE`).
4. Тесты:
   - Добавлены проверки нормализации путей/освобождения ресурсов.
   - Добавлен тест загрузки шейдеров.
   - Добавлен тест иерархии трансформаций сцены.
